This branch is DEPRECATED, may be UPDATED with critical bug fixes. For the latest samples
with Android Studio + gradle experimental plugin, please refer to master branch's 
[other-builds/experimental](https://github.com/googlesamples/android-ndk/tree/master/other-builds/experimental) directory

The samples in this branch known to build with Android Studio 2.2.x and 2.3.x
